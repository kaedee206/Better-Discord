## Autumn Theme for BetterDiscord

A warm, cozy autumn look layered on top of the Translucence base.

### Install (EN)
- Copy `AutumnTheme.theme.css` to your BetterDiscord themes folder.
  - Windows: `C:\Users\<YourUser>\AppData\Roaming\BetterDiscord\themes`
- In Discord → User Settings → BetterDiscord → Themes, enable only `AutumnTheme`.
  - It already imports Translucence, so you don’t need to enable Translucence separately.

## Giao diện Mùa Thu cho BetterDiscord (VI)

Tông ấm mùa thu, xây dựng dựa trên nền Translucence.

### Cài đặt (VI)
- Sao chép `AutumnTheme.theme.css` vào thư mục themes của BetterDiscord.
  - Windows: `C:\Users\<TênUser>\AppData\Roaming\BetterDiscord\themes`
- Vào Discord → User Settings → BetterDiscord → Themes, chỉ bật `AutumnTheme`.
  - Theme đã `@import` Translucence, không cần bật Translucence riêng.

### Lời cảm ơn / Ghi công (VI)
